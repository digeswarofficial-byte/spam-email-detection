# 📧 Spam Email Detection Using Machine Learning

## 📌 Project Overview
This project detects spam emails using Machine Learning and Natural Language Processing (NLP) techniques.

The system classifies emails into:
- Spam Email
- Ham (Normal) Email

## 🚀 Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- NLTK
- Scikit-learn

## 🤖 Machine Learning Algorithm
- Naive Bayes Algorithm

## 📊 Accuracy
✅ Achieved approximately 97% accuracy

## 📂 Project Files
- Spam_Email_Detection.ipynb
- spam.csv
- spam_model.pkl

## 🔄 Project Workflow
Email Input → Data Cleaning → Feature Extraction → Model Training → Spam/Ham Prediction

## 📌 Features
- Automatic spam detection
- High accuracy
- Fast prediction
- Easy implementation
